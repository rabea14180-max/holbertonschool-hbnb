Package Diagram Overview

Presentation Layer:
- Contains API Services.
- Handles client requests and responses.

Business Logic Layer:
- Contains the Facade and core entities.
- Facade acts as a single entry point.
- Entities: User, Place, Review, Amenity.
- Implements all business rules.

Persistence Layer:
- Contains Repositories.
- Manages data storage and retrieval.

Communication Flow:
Client → API Services → Facade → Entities → Repositories
