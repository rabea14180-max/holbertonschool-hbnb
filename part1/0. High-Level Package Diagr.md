This diagram shows a three-layer architecture
Presentation Layer: Handles client requests through API Services.

Business Logic Layer: Contains the core logic and domain entities (User, Place, Review, Amenity) and uses a Facade to simplify interactions.

Persistence Layer: Manages data storage using Repositories.

Requests flow from the API to the Facade, then to the domain entities and repositories, and responses return back to the client.
